# Contributing to Password Cracking Suite

Thank you for your interest in contributing! This project is educational and meant for ethical hacking practices only.

## How to Contribute

1. **Fork** the repository
2. **Create** a feature branch (`git checkout -b feature/YourFeature`)
3. **Commit** your changes (`git commit -m 'Add some feature'`)
4. **Push** to the branch (`git push origin feature/YourFeature`)
5. **Open** a Pull Request

## Code Style

- Use Python PEP 8 style guide
- Add docstrings to all functions
- Include type hints where possible
- Write unit tests for new features

## Areas for Contribution

- [ ] Add more password patterns
- [ ] Improve hash extraction
- [ ] Add real hash cracking (with ethical disclaimers)
- [ ] Create web interface
- [ ] Add performance improvements
- [ ] Improve documentation

## Reporting Issues

Please use GitHub Issues to report bugs or suggest improvements. Include:
- Clear description
- Steps to reproduce
- Expected vs actual behavior
- Python version and OS

## Code of Conduct

This project is for **educational purposes only**. All contributions must adhere to ethical hacking principles and laws.
